Add a new user

Creates a real login for them and grants dashboard access in one step. Share the email/password with them directly — have them change the password after first login (there's no self-service reset page in this build).

People with access

"Revoke" removes their dashboard access immediately. It does not delete their login credentials (that requires the Firebase Admin SDK, which this static-hosted build doesn't have) — so keep passwords private, and treat revoke as "locked out of the app," not "account destroyed."
